## Project Overview
This project demonstrates the use of Python for data extraction, analysis, and visualization in the context of stock market data. Using libraries such as yfinance and web scraping techniques, the project extracts stock price and revenue data for Tesla and GameStop.

---

## Features
- Extraction of stock data using yfinance
- Web scraping to collect revenue data
- Data cleaning and preparation
- Visualization of stock prices and revenues using custom plotting functions
- Analysis performed within Jupyter Notebook

---

## Purpose
The goal of this project is to showcase foundational Python skills applied to real-world financial data, suitable for data science and data analysis portfolios.

---

## How to Use
- Run the Jupyter Notebook to reproduce the data extraction, analysis, and visualization steps.
- Ensure required libraries such as `yfinance`, `pandas`, `matplotlib`, and `beautifulsoup4` are installed.
